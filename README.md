# Simulação Educacional de Phishing – Cibersegurança

## Descrição
Este projeto tem como objetivo estudar o funcionamento de ataques de phishing
em um ambiente controlado, com fins exclusivamente educacionais, conforme
abordado no curso da DIO.

Nenhuma informação real foi coletada ou armazenada.

## Objetivos de Aprendizagem
- Compreender como ataques de phishing são estruturados
- Analisar os riscos desse tipo de ataque
- Reforçar boas práticas de segurança da informação
- Documentar processos técnicos de forma clara
- Utilizar o GitHub para versionamento e apresentação de projetos

## Ferramentas Utilizadas
- Kali Linux (ambiente de laboratório)
- SEToolkit (Security Engineering Toolkit)
- Git e GitHub

## Metodologia
A simulação foi realizada em ambiente local e controlado, utilizando páginas
de teste e dados fictícios, apenas para fins de estudo.

## Considerações de Segurança
O phishing é uma prática criminosa quando utilizada fora de ambientes
autorizados. Este projeto não deve ser reproduzido em ambientes reais.

## Como se Proteger de Ataques de Phishing
- Verificar URLs antes de inserir credenciais
- Não clicar em links suspeitos
- Utilizar autenticação em dois fatores
- Manter sistemas atualizados
